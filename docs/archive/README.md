# Arquivo histórico — documentos superados

Estes documentos são **históricos**. Registram a evolução do projeto Bits
Hierárquicos, mas foram **superados** pelo estudo canônico
[`BH_MASTER.md`](../../BH_MASTER.md) na raiz do repositório.

**Não use estes arquivos como base de decisão.** Em particular,
`arquitetura_bits_hierarquicos.md` (o documento conceitual original) contém
estimativas de hardware que foram depois identificadas como não medidas e
**corrigidas** no estudo sério — essa autocorreção está documentada no
`BH_MASTER.md` (§7, "As autocorreções").

São mantidos por transparência: a evolução de "ideia confundida com codec/banco"
para "arquitetura medida com fronteiras honestas" é parte do mérito do trabalho.
